GEO Implementation Notes

This workshop distills Prof. Fuchs' 5-step GEO playbook into a culinary context.

  1. Build authority where AI looks: cite consortia, labs, and guild data.
  2. Make content API-able: semantic HTML, JSON-LD, and consistent ingredient tables.
  3. Engineer fact density: every section answers who/what/when with numbers.
  4. Cover the full query journey: summary cards, FAQs, and comparative links.
  5. Maintain technical hygiene: llms.txt, IndexNow hooks, lightweight HTML.

Each recipe page doubles as an answer asset: metrics, comparisons, and schema markup ready for generative engines.
