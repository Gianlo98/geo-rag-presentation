# GEO & RAG Presentation

Presentation about **Generative Engine Optimization (GEO)** and **RAG (Retrieval-Augmented Generation)** by Gianlorenzo Occhipinti for the Learning Week Block on November 17th, 2025.

## About

This presentation explores the concepts and applications of:
- Generative Engine Optimization (GEO)
- Retrieval-Augmented Generation (RAG)
- Their intersection and practical implementations

## Getting Started

### Prerequisites

- Node.js >= 18.0.0

### Installation

```bash
npm install
```

### Running the Presentation

```bash
npm start
```

The presentation will be available at [http://localhost:8000](http://localhost:8000)

### Building

```bash
npm run build
```

## Files

- `index.html` - Main presentation file
- `demo.html` - Full demo with reveal.js examples
- `examples/` - Example presentations demonstrating features

## Technology

This presentation is built using [reveal.js](https://revealjs.com/) - an open source HTML presentation framework.
